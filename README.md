# Sea Level Time Series Visualizer

This project visualizes historical sea level data using Python. It generates three types of plots:

- **Scatter Plot** – Shows historical sea level measurements.  
- **Line of Best Fit (1880–2049)** – Projects trends based on all available data.  
- **Line of Best Fit (2000–2049)** – Projects trends based on recent data for comparison.  

The visualizations are saved as `sea_level_plot.png`.
